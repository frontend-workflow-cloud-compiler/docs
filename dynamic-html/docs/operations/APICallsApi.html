This is the API

cssGet

URL

https://virtserver.swaggerhub.com/TheMaaarc/Unglue/1.1.1/css

HTTP Method

GET

Response Type

CompileResponse

Parameters

  • Query: files List(files)

    Alle input files that will be needed for the compilation.

  • Query: distFile String

    The name of the dist file.

  • Query: mainFiles List(String)

    The \"entry\" .scss files (generally the ones without _ infront of their name.)

  • Query: options options

    The options that should be used for compilation.
